# SPELLS DIRECTORY

A premium, magazine-style editorial directory for verified spell caster services.

## Design Aesthetic

Inspired by editorial magazine layouts with a neo-brutalist, boxy design approach featuring:
- **Beige** (#E8D5C4) - Primary background color
- **Black** (#1A1A1A) - Text and borders
- **Red** (#C41E3A) - Accent color for emphasis

## Features

- **Editorial Magazine Layout**: Bold typography and thick borders throughout
- **4x4 Grid System**: 16 spell casters displayed in a clean grid layout
- **Boxy Design**: Strong geometric layout with prominent borders
- **Fully Responsive**: Adapts to all screen sizes (desktop, tablet, mobile)
- **Professional Typography**: Bold, condensed fonts for impact

## Structure

### Header
- Site title with premium branding
- Tagline and establishment details

### Hero Section
- Large title and description
- Statistics boxes showing verified casters and satisfied clients

### Main Grid (4x4)
Each card includes:
- Small placeholder image
- Practitioner name
- Description of services
- Tags (e.g., "LOVE SPELLS", "TAROT READING")

### Footer
- Contact information
- Social media links
- Copyright information

## Color Palette

- **Beige**: #E8D5C4 (primary background)
- **Black**: #1A1A1A (text, borders)
- **Red**: #C41E3A (accents, highlights)
- **White**: #FFFFFF (card backgrounds)

## Typography

- Headings: Arial Black (900 weight)
- Body: Arial (400 weight)
- All caps for titles and labels

## Responsive Breakpoints

- **Desktop**: 4 columns (1600px+)
- **Large Tablet**: 3 columns (768px - 1200px)
- **Tablet**: 2 columns (480px - 768px)
- **Mobile**: 1 column (<480px)

## Browser Support

Works on all modern browsers including Chrome, Firefox, Safari, and Edge.

## Customization

Replace placeholder images by updating the `.card-image` content in HTML. Add real images inside the `.card-image` divs to showcase actual spell casters.
